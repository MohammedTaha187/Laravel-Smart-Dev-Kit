# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-15
### Added
- Initial release of the Ultimate Modular SaaS Engine.
- Modular architecture with `nwidart/laravel-modules`.
- Multi-tenancy with `stancl/tenancy`.
- Type-safe DTOs with `spatie/laravel-data`.
- Stateless JWT Authentication.
- Multi-language support with `spatie/laravel-translatable`.
- Smart CRUD Generator with Service/Repository/Contract support.
- Horizon (Queue Worker) and Scheduler services in Docker.
- GitHub Actions CI workflow for automated testing.

### Fixed
- README image path exposure.
- Dependency version pinning for stability.
- Generator namespace issues in Modules.
