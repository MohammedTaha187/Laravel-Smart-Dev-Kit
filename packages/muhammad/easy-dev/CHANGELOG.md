# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-17

### Added
- Professional CRUD Generation (`smart:crud`) with 12-file generation logic.
- Interface-Service-Repository mandatory flow by default.
- Spatie Data (DTO) and API Resource integration.
- Database relationship auto-detection (`smart:sync-relations`).
- Feature generation from migrations (`smart:from-migration`).
- BaseRepository and BaseServiceInterface for predictable architecture.
- Full Pest integration testing.
- Modular support for `nwidart/laravel-modules`.
- Translatable fields support via Spatie Translatable.
- Media support via Spatie Media Library.
- Activity logging support via Spatie Activity Log.
